PHPhotoalbum

PHPhotoalbum was a web based photo album I created in 2002. It was maintained until 2004 and received around 106.000 downloads. It was later used as the foundation for Coppermine Photo Gallery which has grown to be a very popular open source project and is still under active development.

PHPhotoalbum is no longer maintained. Please get in touch if you have any questions.
